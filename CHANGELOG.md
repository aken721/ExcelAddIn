# 更新日志 | Changelog

所有重要版本的变更记录均在此文件中。格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/) 规范。
All notable changes are documented here, following the [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) convention.

## [v2.5.1.0] - 2026-04-21

### ✨ 新增 | Added
- **Skills 架构**：将插件内全部功能封装为 Skills，支持 LLM 通过自然语言直接调用插件功能
- **品牌升级**：项目正式更名为万象插件（TableMagic）

### 🔧 优化 | Changed
- 对话框气泡显示逻辑优化：
  - 修复气泡高度与文字内容不协调的问题
  - 优化气泡框与文字的同步显示，消除首次渲染延迟
  - 优化流式输出时等待气泡的切换逻辑，内容开始输出时立即移除等待状态

---

## [v2.5.0.0] - 2026-03-23

### ✨ 新增 | Added
- Skills 扩展架构重构：模块化设计，每个技能独立封装，支持动态加载，工具定义与执行逻辑统一管理
- 支持 Ollama、LM Studio、vLLM、llama.cpp 等本地模型，可自动获取模型列表
- 多模型提供商支持：原 DeepSeek 单一绑定改为多模型自由选择

### 🔧 优化 | Changed
- AI 对话响应速度大幅提升
- 对话气泡显示优化，修复容器与内容延迟分开显示的问题
- 思考占位符优化：思考完成后在同一容器内显示回复内容
- 去除"优先使用提示工程模式"手动选项，改为自动判断（不支持 Function Calling 的本地模型自动切换为提示工程模式）
- 使用动态 JSON 解析，兼容不同模型提供商的响应格式
- 扩展关键词映射，覆盖更多自然语言表达
- 添加智能工具选择调试日志，修复关键词匹配冲突问题

### 🐛 修复 | Fixed
- 修复参数名称映射问题，支持模型使用不同参数名调用工具
- 添加详细的错误提示（API Key 错误、网络错误、地址错误等）

## [v2.4.7.0] - 2025-12-25

### ✨ 新增 | Added
- 图表增强新增动态图表生成：支持 HTML 交互式动态图表和 GIF 动图图表（当前支持条形图和柱状图）
- 批量生成 Word 文档新增生成范围配置：可选择生成全部行数据或仅选中行数据

### 🔧 优化 | Changed
- 图表增强数据区域支持从 Excel 表格中直接获取选中区域
- 批读文件名完善"文件/目录"和"包含子目录"选项的读取逻辑：修改选项后自动刷新 `_rename` 表内容
- 优化部分 UI 界面，修复若干已知问题
